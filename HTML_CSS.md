# HTML_CSS

HTML/CSS基礎技術

## 概要

[[HTML]]（HyperText Markup Language）と[[CSS]]（Cascading Style Sheets）は、Webページを構築するための基本的な技術です。

## HTML

Webページの構造を定義するマークアップ言語。要素、タグ、属性を使ってコンテンツの意味と構造を表現します。

## CSS

Webページの見た目やレイアウトを制御するスタイルシート言語。色、フォント、配置などのデザインを定義します。

## 主要な概念

- **セマンティックHTML** - 意味のあるタグを使用してコンテンツを構造化
- **レスポンシブデザイン** - デバイスサイズに応じた柔軟なレイアウト
- **Flexbox/Grid** - 現代的なレイアウトシステム
- **CSSセレクター** - 要素を選択してスタイルを適用
- **メディアクエリ** - 画面サイズに応じたスタイル切り替え
